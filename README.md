# myhome 만들기



# [프로젝트소개]

Java + html, js, jsp, servlet 등을 활용하여 동적 홈페이지를 구상해보자!



### 주요 기능

- 관리자 & 일반 회원 공통 기능

  - 로그인 / 로그아웃
  - 회원정보 수정 / 탈퇴
  - 게시글 조회 / 작성 / 수정 / 삭제
  - 파일 업로드 / 다운로드

- 관리자 페이지 기능

  - 회원 강퇴

  

#### 개발 환경

JDK 1.8

IDE : eclipse
